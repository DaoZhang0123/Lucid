1. AUTONOMOUS CONTROL
   Lucid uses a Large Language Model (LLM) to autonomously drive your mouse,
   keyboard, and clipboard. It can open any app, send messages, read and
   write files, and access the network on your behalf. The LLM may misread
   the screen, click the wrong button, send the wrong message, or delete the
   wrong file. You are solely responsible for every action Lucid performs
   under your account and for every resulting consequence.

2. DATA TRANSMISSION
   Lucid sends screenshots, window titles, and your task instructions to the
   LLM provider you have configured (Anthropic, GitHub Copilot, an
   OpenAI-compatible proxy, etc.). Screenshots may contain personal
   information, chat history, password fields, bank card numbers, and
   confidential work content. Do not use Lucid while sensitive information is
   visible on screen.

3. EMERGENCY STOP
   Press Ctrl+Alt+Esc at any time to immediately stop the current task and
   reclaim input control. You can also force-quit from the system tray icon.

4. NOT SUITABLE FOR
   - Online banking, payment passwords, or fund transfer confirmations.
   - Communication that involves another person's privacy, unless consented.
   - Any operation with irreversible consequences (deleting databases,
     git push --force, publishing on social media, sending contracts, etc.)
     - use only under direct supervision.
   - Windows that require administrator privileges (UAC, Group Policy) are
     already blocked.

5. PROVIDER ACCEPTABLE USE
   You must use your own LLM API key and comply with the provider's
   Acceptable Use Policy (Anthropic AUP, OpenAI Usage Policies, GitHub
   Copilot Terms, etc.). Lucid does not relay, store, or proxy your API
   credentials.

7. LOCAL LAW
